# API 接口文档模板

## 📋 模板简介

API 文档是前后端协作的契约，也是第三方集成的接口说明。本模板提供标准的 RESTful API 文档格式。

**当前版本**: v1.0.0

---

## 🎯 使用场景

- 前后端接口对接
- 第三方 API 集成
- 移动端 API 说明
- OpenAPI/Swagger 补充文档

---

## ✅ 最佳实践

1. **保持同步**: API 变更后及时更新文档
2. **提供示例**: 每个接口都有请求和响应示例
3. **说明错误**: 列出所有可能的错误码
4. **版本管理**: 使用 URL 版本或 Header 版本
